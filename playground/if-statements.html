Truthy Values

1, -10, all other numbers are truthy

An full string is a truthy value

A string of zero "0" is truthy (as long as the string isn't empty "" it is truthy)

An empty array [] is truthy

An empty object {} is truthy

Flasy Values

0 is a falsy value

Undefined is a falsy value

Null is a falsy value

NaN is a falsy value

An empty string is a falsy value
